# Simulation Fidelity Directive
> Binding. Supersedes all prior "content count" metrics.
- Stop measuring success by JSON count, biome count, species count.
- Start measuring by: every location feels alive, every NPC has behavior, every beast is in a food web, every item has history.
- Every location needs ~15 simulation layers.
- Every settlement must FUNCTION (NPC schedules, daily routines, political conflicts).
- Food web: populations not species, predator-prey chains, resource-driven dynamics.
- Civilizations simulate trade, wars, politics, markets.
